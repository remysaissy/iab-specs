# iab-specs-registry_agent

Registry Agent 1.0 support for the [iab-specs](https://crates.io/crates/iab-specs) ecosystem.

## Overview

Implements the Registry Agent 1.0 specification for agent discovery and trust lifecycle management.

- **Agent registration** — RegisteredAgent with capabilities and trust scores
- **Trust lifecycle** — TrustStateMachine with validated state transitions
- **Registry search** — RegistrySearchFilter for agent discovery queries
- **Trust events** — TrustEvent tracking for audit and compliance
