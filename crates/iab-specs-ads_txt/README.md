# iab-specs-ads_txt

Ads.txt 1.1 support for the [iab-specs](https://crates.io/crates/iab-specs) ecosystem.

## Overview

Provides typed Rust data structures for parsing and generating Ads.txt 1.1 files, including:

- **AdsTxt** — Root container for ads.txt entries with contact and owner domain
- **AdsTxtSystem** — Individual authorized seller entries
- **SellerRelationType** — DIRECT or RESELLER relationship types
- **ManagerDomain** — Manager domain declarations
