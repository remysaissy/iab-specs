# iab-specs-buyer_agent

Buyer Agent 1.0 support for the [iab-specs](https://crates.io/crates/iab-specs) ecosystem.

## Overview

Implements the Buyer Agent 1.0 specification for demand-side autonomous agents in programmatic advertising.

- **Campaign planning** — Campaign lifecycle with state machine management
- **Deal negotiation** — Deal proposals, counter-offers, and booking workflows
- **UCP embeddings** — Universal Campaign Parameters for embedding transport
- **State machines** — CampaignStateMachine and DealStateMachine with validated transitions
